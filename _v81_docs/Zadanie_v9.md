# ВЫВОД 🧠

Природа: сборник технического анализа. Один литературный абзац ≥ 300 слов. Объединяет результаты всех разделов в развёрнутый аналитический текст.

Стиль: деловой литературный язык, понятный трейдеру и новичку. ТА запрещён ТОЛЬКО в буллитах.

Начало абзаца: «Сейчас [актив] (#тикер) торгуется по [цена].» Далее краткая история прихода (упали от..., выросли от..., стоим на уровне N дней).

## Содержание абзаца

• Контекст: актив и текущая цена

• Обоснование каждого шага маршрута

• Манипуляция (если вероятна) — куда, глубина, обоснование

• Первый шаг — обоснован причиной

• Характер движения (импульс / манипуляция / сползание)

• Условие слома структуры

## Буллиты (7 штук)

Формат: название: цена (% от текущей).

Маршрут: 🟢 цена (+%) 🔴 цена (−%) — с фазами в скобках (импульс / откат / добой / манипуляция / финал).

📌 ключевые цели: цена (%), цена (%)

📌 остальные цели: цена (%), цена (%)

📌 вероятный маршрут: 🟢 цена (+%, фаза) 🔴 цена (−%, фаза) ... — основное направление от текущей цены, включая манипуляции, откаты, добои

📌 вероятные сроки: N–M дней

📌 приостановка и наблюдение: цена (%) — [пробой первой границы]

📌 слом структуры и пересмотр: цена (%) — [закрепление за последней границей]

В буллитах ТА запрещён. Только цены, проценты, метки 🟢/🔴.
