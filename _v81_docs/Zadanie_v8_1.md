ПРАВИЛО ОФОРМЛЕНИЯ РАЗДЕЛОВ

Каждый раздел выполняется ПО КАЖДОМУ ТФ из предоставленных данных

Каждый ТФ = один абзац внутри раздела

Несколько ТФ = несколько абзацев внутри раздела (1D, 4H, 1H — каждый отдельным абзацем)

Раздел завершается ОДНИМ общим выводом по всем ТФ — итог анализа раздела целиком

Все цены — в формате 2.640 (+0.94%) от текущей цены

Зоны (цели <0.3%) — формат: зона X–Y

Пропуск ТФ в разделе = нарушение. Если данных нет — указать «ТФ не предоставлен»

5 📉 ДИВЕРГЕНЦИИ

Задачи по каждому таймфреймуАнализ наличия всех видов дивергенций по RSI, CVD, MACD, Stochastik

RSI: тип дивергенции, зона (>70 / <30 / нейтральная). Поддерживает ли экстремумы

CVD: дивергенция с ценой

Стохастик (%K/%D): зоны, кроссы, дивергенции

MACD: направление, гистограмма, нулевая линия, дивергенции

Аналог: По скриншоту: считай значения RSI, CVD. Определи дивергенции визуально по расхождению экстремумов цены и индикатора

Цифра: RSI дивергенции из CSV (Regular/Hidden Bullish/Bearish) + собственный расчёт. Стохастик (%K, %D), MACD, Signal, Histogram — из CSV

6 📊 УРОВНИ

Задачи по каждому тфаймфрейму

Задача: определить 5 важнейших и сильнейших уровней сопротивления и 5 уровней поддержки по скриншоту или CSV 

Источники: ТОЛЬКО пивоты (все комплекты), значимые свинги, исторические уровни реакций, круглые числа

Учитывать: многократные касания, развороты, длинные хвосты, всплески объёма

Формат: R2 — 3.297 (+2.14%)

Аналог: По скриншоту: считай все пивоты. Найди исторически сильные уровни — там где ЧАСТО происходили реакции цены (развороты, длинные хвосты, всплески объёма, многократные касания)

Цифра: Программно: рассчитай пивоты (дневные, недельные, месячные + квартальные если ТФ ≥ 1D) по формулам: P=(H+L+C)/3, R1=2P−L, R2=P+(H−L), R3=H+2(P−L), R4=P+3(H−L)/2, S1=2P−H, S2=P−(H−L), S3=L−2(H−P), S4=P−3(H−L)/2. По закрытым периодам. Исторически сильные уровни — значения цены, где рынок ЧАЩЕ всего бывал и ПРОВОДИЛ больше времени (распределение цены по уровням, максимум касаний и время консолидации)

8 🔊 VSA

Задачи по каждому тфамфреймупровести VSA анализ

Оцени усилие vs результат (V/L ratio)

Найди аномалии: Climax, No Demand, No Supply

Определи направление CVD: кто давит рыночными ордерами

Аналог: По скриншоту: оцени объёмные бары визуально, определи аномалии, считай CVD

Цифра: Рассчитай V/L ratio, OBV, найди бары >2σ от среднего за 50 баров

кульминация.– Даётся вывод: активность покупателя / продавца.⬛️ Оценивается эффективность усилия через отношение объёма к результату (V/L).⬛️ Рост V/L = на единицу движения тратится больше объёма = сценарий слабеет.⬛️ Если объём растёт, но движение замирает — усиление считается неэффективным.⬛️ Используется как фильтр вероятной фиксации или приостановки.9 📊 ОБЪЁМНЫЕ ЗОНЫ

12 🔄 ТЕМП РЫНКА

Задачи:

Рассчитай ATR текущего ТФ (пункты и %)

Рассчитай дневной ATR(14) (пункты и %) — обязательно, даже если ТФ не дневной

Рассчитай k-темпа: k = Mean(TR, 10–20 баров) / ATR(ТФ)

Интерпретируй k: <0.8 замедлен / 0.8–1.2 норма / >1.2 перегрет

Рассчитай сроки достижения ключевых целей: ceil(Dist% / (k × ATR_дн% × F)). F: 1.0 тренд / 0.7 коррекция / 1.3 пробой

Аналог: По скриншоту: считай ATR из индикатора на графике

Цифра: Программно: рассчитай ATR(ТФ), ATR(D), k-темпа, сроки

Цифра: Программно: FVG по формуле. Гэпы по OHLC. Ликвидность по swing-точкам

Природа Вывода

Вывод = аналитическая заметка технического анализа. Научный текст с доказательной базой всех движений. Все данные разделов, все значения индикаторов, все обоснования — объединены в развёрнутый аналитический текст. Приоритет — развёрнутый, пояснительный, доказательный вывод, объединяющий результат работы аналитика по ВСЕМ разделам.

Абзац

Длина: ≥ 300 слов.

Как формируется: из выводов каждого раздела (вывод× 13–18 разделов)+ описание прогнозируемого маршрута подробно и обосновано. Связать в единый абзац прозы.

Стиль: деловой литературный язык

Начало абзаца ВСЕГДА: «Сейчас [название актива] (#тикер) торгуется по [цена].» Далее — краткая история прихода к текущей цене (упали от..., выросли от..., стоим на уровне N дней и т.д.).

ТА в абзаце: абзац СТРОИТСЯ на инструментах ТА — RSI, волны, Фибо, ALMA, Вайкофф, CVD. ТА запрещён ТОЛЬКО в буллитах.

Содержание абзаца

Контекст: актив и текущая цена

Обоснование каждого шага маршрута

Манипуляция (если вероятна) — куда, глубина, обоснование

Первый шаг — обоснован конкретной причиной (закончить волну, отретестить пробитый уровент

Характер движения (импульс / манипуляция / сползание)

Условие слома структуры

Буллиты (6 штук)

Формат каждого буллита: название:  ТОЛЬКО цена (% от текущей). Списком.

Маршрут: 🟢 цена (+%) 🔴 цена (−%) 

📌 ключевые цели: цена (%), цена (%)

📌 остальные цели: цена (%), цена (%)

📌 вероятный маршрут: текущая цена(%)->🟢 цена (+%)-> 🔴 цена (−%) ... — основное направление от текущей цены, включая манипуляции, откаты, добои, ретесты

📌 вероятные сроки: в часах/днях/неделях (сроки достижения последней цели в маршруте)

📌 отмена маршрута: уровень цена (%)

В буллитах ТА запрещён. Только цены, проценты, метки 🟢/🔴. Запрещено ссылаться на документы проекта. После вывод спросить нужен Фомо?



ФОМО-  это формат публикации для ТГ канала, делается по запросу Босса.
