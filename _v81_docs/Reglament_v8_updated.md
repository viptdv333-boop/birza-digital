VII. КЛАССИФИКАЦИЯ ЦЕЛЕЙ

Ключевые

Важные по структуре и подтверждённые несколькими разделами. Скелет маршрута. ТОЛЬКО ключевые цели могут быть конечными целями маршрута и уровнем слома.

Остальные

Все прочие цели из разделов. Уплотнение и детализация маршрута. Не могут быть конечной целью или уровнем слома.

Правила

ВСЕ цели (ключевые + остальные) ИЗ ВСЕХ РАЗДЕЛОВ обязательно входят в маршрут

Маршрут полностью состоит из целей, выявленных в анализе — никаких выдуманных точек

Количество целей в маршруте = количество целей в буллитах «ключевые + остальные»

Уровень слома — ОБЯЗАТЕЛЬНО одна из целей маршрута (крайняя в противоположном направлении). Слом = закрепление за этим уровнем

Цели <0.3% → зона X–Y

Маршрут: 8–15+ смысловых узлов. Больше допустимо, если все узлы подтверждены разделами

ЗАПРЕЩЕНО: маршрут короче списка целей. Пример ошибки: 5 ключевых + 5 остальных = 10 целей в списке, но маршрут из 6 точек. Все 10 должны быть в маршруте.

Порог подтверждения:

• Ключевые — подтверждены минимум 3 инструментами (волны, фибо, уровни, объём, профиль, RSI и др)

• Второстепенные — подтверждены 1–2 признаками, но встроены в структуру маршрута

• Крайние границы зон продавца/покупателя включаются как возможные цели даже при 1 инструменте, если соответствуют расширению / хвосту / волне / уровню / ликвидности / стопам / POC

• Маркировка: «возможен финал к 60.00 (−5.2%)», «возможна манипуляция к 68.80 (+2.3%)»

VIII. МАРШРУТ

🔴 МАРШРУТ ВСЕГДА НАЧИНАЕТСЯ С ТЕКУЩЕЙ ЦЕНЫ. Первая точка маршрута = цена в шапке анализа. Запрещено начинать с уровня, до которого ещё надо дойти.

Принцип

Строй маршрут на основании технического анализа, структуры рынка, своих знаний и опыта. ПРАВИЛО ПОСТРОЕНИЯ: старший ТФ — главный, задаёт направление и ключевые цели. Младшие ТФ — уточняющие, делают маршрут подробнее без изменения сценария старшего. Используй выводы разделов: основное направление, уровни и цели, вероятная манипуляция или откат перед началом основного движения. В маршруте учитывай возможные откаты и ретесты сильных уровней. Маршрут — это не зигзаг через все цели подряд, а логическая цепочка движения цены по структуре.

Маршрут ВСЕГДА начинается с текущей цены.

9 шагов построения

1. Собери ВСЕ цели из всех выполненных разделов

2. Классифицируй: ключевые / остальные. Применяй k-темпа

3. Определи направление: где старшая ключевая цель

4. Проверь манипуляцию по чек-листу. 3+ признака = вероятна. Включи в маршрут с пометкой, направлением и глубиной

5. Определи первый шаг по структуре. Обоснуй причиной

6. Построй маршрут-зигзаг через ВСЕ выявленные цели. Маршрут начинается с текущей цены. Каждый разворот обоснован структурой. Единая ветка. Количество точек = количество целей в буллитах

7. Определи уровень слома: ключевая цель из маршрута, закрытие дневки за ней = слом. Крайняя цель в направлении, противоположном финальной

8. Очисти: цели <0.3% → зона X–Y

9. Рассчитай сроки: суммарная дистанция ÷ (k × ATR_дн% × F)

Антипаттерн

❌ Линейная лестница: 100 → 105 → 110 → 115 → 120 — цена так не ходит

❌ Зигзаг через ВСЕ цели подряд без учёта структуры — хаотичное метание между уровнями

✅ Логическая цепочка по структуре: 🟢 108 (+8%, импульс до ключевого сопротивления) 🔴 103 (+3%, откат/ретест) 🟢 115 (+15%, продолжение) 🔴 109 (+9%, коррекция) 🟢 120 (+20%, финальная цель)

Между ключевыми уровнями — откат или ретест ТОЛЬКО если его даёт структура. Не выдумывать откаты ради красивого зигзага.

Чек-лист первого шага

Фаза: завершена iii → откат. Завершена ii → импульс. UTAD → вниз. BU → вверх

ALMA 200: цена под ALMA + наклон вниз → шорт. Над ALMA + вверх → лонг. Плоская → оцени по RSI/CVD

RSI: перегрет → откат. Нейтральный → тренд продолжается. Дивергенция → противоположная сторона

Ликвидность/FVG: магнит сверху ≤0.6×ATR → вверх. Снизу ≤0.6×ATR → вниз. Оба → решает фаза и ALMA

VSA: объём с результатом → подтверждает. Кульминация без движения → разворот. Спринг → вверх. UTAD → вниз

Кромка: оттолкнулась от S с объёмом → вверх. Отказ от R → вниз. Фейк + быстрый возврат → истинное направление

Формула: Фаза + ALMA + RSI + CVD + VSA + Ликвидность → большинство в одну сторону = первый шаг

Правило: Первый шаг — реальный ближайший шаг по графику, а не желаемое направление. Маршрут не может начинаться с пробоя уровня, до которого ещё надо дойти.

Признаки манипуляции (3+ = вероятна)

Раздел 11: скопление стопов рядом с текущей ценой (≤1×ATR) — ликвидность для сбора

Раздел 13: незаполненный FVG или гэп в противоположном направлении от основной цели

Раздел 10: фаза SPRING (ложный пробой вниз) или UTAD (ложный пробой вверх) по Вайкоффу

Раздел 8: аномальный объём без движения цены (Climax) — крупный игрок набирает позицию

Раздел 4: длинные тени, ложные пробои уровней, выносы за хай/лой и возврат

Раздел 5: дивергенция RSI или CVD с ценой — движение не подтверждено

Действие: если вероятна — указать направление (куда поведут), глубину (до какого уровня), обоснование (какие признаки), включить в маршрут с пометкой «возможна манипуляция».

Уровень слома

Уровень слома — всегда КЛЮЧЕВАЯ цель из маршрута. Не произвольная цена

Крайняя цель в направлении, противоположном финальной цели. Финальная внизу — слом сверху. Финальная вверху — слом снизу

Закрытие дневной свечи за уровнем = слом

Обоснование обязательно: почему именно эта цель и что произойдёт при пробое

Пример: цена 100, старшая цель 120. Слом — 92 (−8%): ключевая цель маршрута. Закрытие дневки ниже 92 — прогноз на 120 аннулируется.

Два уровня отмены

Слом структуры и пересмотр: по закреплению за последней границей зоны продавца/покупателя

Пробой + закрепление = слом. Пробой без закрепления ≠ слом.

Маршрут не обрезается. До цели в горизонте ТФ, дальше — с пометкой «при продолжении».

Чек-лист первого шага

• 1️⃣ Структура: у сопротивления / перегрев RSI / замедление ALMA → сначала вниз. От поддержки / рост RSI / ускорение ALMA → вверх

• 2️⃣ Фаза: завершена iii → откат. Завершена ii → импульс. BU → вверх. UTAD → вниз

• 3️⃣ ALMA/RSI: теряет угол / RSI вниз → откат. Ускоряется снизу / RSI в импульс → рост

• 4️⃣ Зоны: ликвидность / FVG / POC под ценой → сначала вниз. Сверху → сначала вверх

• ❌ Ошибка = несоответствие первого шага текущей фазе и поведению графика

Методология построения сценария

• Старт всегда от текущей цены

• Определяется активная сторона по RSI / ALMA / объёму

• Анализируется фаза (волна, Вайкофф, структура)

• Уточняется ближайшая цель

• Проверяется возможная манипуляция, поведение в зоне

• Если структура допускает откат — он закладывается

• Каждая фаза берётся с того ТФ, где она подтверждена

XIII. СРОКИ ДОСТИЖЕНИЯ

«Цель достижима до конца текущей недели»

• Пробой зоны = отмена только при закреплении

Пробой без объёма

• Пробой зоны без объёма и сигнала = манипуляция

• Такая свеча не даёт права на отмену

• Закрепление без объёма = временная приостановка, не структурная отмена

• Эл обязан указать: «Пробой зоны без подтверждённого объёма. Вероятна ловушка / манипуляция»
