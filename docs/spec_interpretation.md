### 1.2 Посчитай

```
UP: 7 сигналов
DOWN: 15 сигналов
→ Направление: DOWN
```

**Правило: большинство = направление. Без исключений.**

Не взвешивай «важность» отдельных сигналов. Один сигнал = один голос. Дивергенция RSI не «важнее» ALMA-веера. Считай и бери большинство.

**Единственное исключение:** если ≥3 НЕЗАВИСИМЫХ дивергенции (RSI + CVD + MFI — три разных индикатора показывают одно и то же) + перепроданность/перекупленность стохастика + кульминация объёма — это кумулятивный блок, который весит как 5 голосов, а не 3. Потому что три независимых источника, подтверждающих одно и то же, — это не три отдельных мнения, а один сильный факт, измеренный тремя способами.

## ШАГ 3: СКЛЕЙ БЛИЗКИЕ УРОВНИ В ЗОНЫ

```
Если два уровня ближе 0.3% друг к другу → объединить в зону.
Зона = нижний–верхний. Название = перечисление через «/».
```

Пример:
```
Fibo 1.272 (2.622) + M.S1 (2.624) + W5=W3 (2.607) → ЗОНА 2.607–2.624 (-3.06%...-2.99%)
Это сильный кластер — 3 источника. → Ключевая цель.
```

## ШАГ 4: КЛАССИФИЦИРУЙ

```
Ключевые:      3+ источника подтверждают один уровень/зону
Второстепенные: 2 источника
Остальные:      1 источник, но структурно важный (экстремум, пивот, POC)
```

Отбрось уровни без структурного смысла (MC-кластер без совпадения с TA, одиночный FVG размером 0.001).

## ШАГ 5: ОПРЕДЕЛИ ПЕРВЫЙ ШАГ

Приоритет определения:

**1. Proximity FVG** (раздел 13): если есть открытый FVG в пределах 0.6 × ATR — цена идёт к нему первым. Медвежий FVG над ценой → первый шаг вверх. Бычий FVG под ценой → вниз.

**2. Ликвидность** (раздел 11): если нет FVG в proximity — смотри, где ближайший пул стопов. Стопы под минимумом → вниз. Стопы над максимумом → вверх.

**3. Общее направление** (шаг 1): если нет ни FVG, ни очевидной ликвидности — первый шаг = общее направление прогноза.

**Важно:** первый шаг может быть ПРОТИВ общего направления. Если прогноз «вверх», но ближайшая ликвидность под ценой — первый шаг вниз (задёрг к стопам), затем разворот и движение вверх. Это нормально. Это манипуляция.

## ШАГ 6: ПРОВЕРЬ МАНИПУЛЯЦИЮ

Посмотри на данные последних 10–20 баров:

| Признак | Как определить |
|---------|---------------|
| Длинный хвост + возврат | Tail spike (раздел 11) + цена вернулась в ≤3 бара |
| Высокий объём без продолжения | Аномалия объёма (раздел 8) + следующие бары не продолжили направление |
| Нет закрепления за уровнем | Пробой уровня → но нет 2 закрытий за ним И хода ≥ 0.2×ATR |
| Пробой в FVG + возврат | Цена зашла в зону FVG и вернулась обратно |
| Failure-импульс | Резкий бар > 2σ range → немедленный откат |
| Ход против старшего тренда | Направление последних баров противоречит swing-структуре |

**Правило: ≥2 признака → манипуляция ВОЗМОЖНА. ≥4 → манипуляция ВЫСОКАЯ.**

Если манипуляция обнаружена:
- Определи направление задёрга (против основного хода)
- Определи зону задёрга (ближайший пул стопов в направлении задёрга)
- Маршрут начинается: S0 → задёрг к зоне → разворот → основной ход

## ШАГ 7: ПОСТРОЙ МАРШРУТ

### Если манипуляция НЕТ:

```
Текущая цена → ближайшая цель в направлении прогноза → следующая по цене → ... → дальняя
```

Просто иди по отсортированным целям в направлении прогноза. Каждая точка маршрута = уровень из таблицы шага 2.

**Пример (прогноз вниз, цена 2.705):**
```
2.705 → 2.698 (swing min, -0.26%) → 2.674 (D.S1, -1.15%) → 2.656 (W.S2, -1.80%) → 2.622-2.624 (кластер M.S1/Fibo, -3.06%) → 2.564 (волновая цель, -5.21%) → 2.526 (Fibo 1.618, -6.63%)
```

### Если манипуляция ВОЗМОЖНА/ВЫСОКАЯ:

```
Текущая цена → задёрг к зоне стопов (против прогноза) → разворот → ближайшая цель по прогнозу → ... → дальняя
```

**Пример (прогноз вверх, манипуляция вниз, цена 2018):**
```
2018 → 2002 (задёрг к стопам под минимумом, -0.82%) → разворот → 2039 (POC, +1.00%) → 2052 (W.R1, +1.63%) → 2066 (VAH, +2.36%) → 2092 (ретест максимума, +3.65%)
```

### Правила маршрута:

1. **Одна ветка.** Никаких «если удержит → вверх, если пробьёт → вниз».
2. **Каждая точка = уровень из таблицы.** Нельзя ставить произвольные цены.
3. **Зоны (<0.3%) — одна точка.** Не засоряй маршрут.
4. **Минимум 5 целей, максимум 10.** Если целей больше — оставь ключевые и второстепенные, убери остальные из маршрута (но не из списка целей).
5. **Все цели из трёх категорий должны существовать.** Маршрут включает ключевые обязательно, второстепенные по пути, остальные в списке.

## ШАГ 8: ОПРЕДЕЛИ СРОКИ

**Формула:**
```
Дней = ceil(Dist% / (k × ATR_дневной% × F))
```

Где:
- Dist% = расстояние до цели в %
- k = k-темпа из раздела 12
- ATR_дневной% = дневной ATR в % от цены
- F = 1.0 по тренду, 0.7 коррекция/против тренда, 1.3 пробой/ускорение

**Считать до ПОСЛЕДНЕЙ цели маршрута.**

Конвертация:
- < 24 часов → «~X ч»
- ≥ 24 часов → «~X дн»

## ШАГ 9: ОПРЕДЕЛИ УРОВЕНЬ СЛОМА

**Правило:** слом = ближайшая структурная точка, закрепление за которой аннулирует сценарий.

**Как выбрать:**

Если прогноз ВНИЗ:
- Возьми последний LH (Lower High) из swing-структуры ZigZag 5%
- Или потолок последней консолидации
- Или фрактальный максимум последних дней
- Выбери БЛИЖАЙШИЙ из них к текущей цене
- Проверь: расстояние не дальше ~1.5 × ATR дневной. Если дальше — возьми ближе

Если прогноз ВВЕРХ:
- Возьми последний HL (Higher Low) из swing-структуры
- Или дно последней консолидации
- Или критическую волновую границу (напр. основание волны 1)
- Выбери БЛИЖАЙШИЙ

**Обоснование слома должно быть конкретным:**
❌ «2.977 — уровень слома» (почему именно он?)
✅ «2.738 (+1.22%) — потолок консолидации 08-09.04; закрепление выше аннулирует пробой и подтверждает разворот»

## ГЛАВНЫЕ ПРИНЦИПЫ (запомнить)

1. **Считай голоса. Бери большинство.** Не думай «но этот сигнал важнее». Нет. Один сигнал = один голос.

2. **≥3 дивергенции из разных источников = кумулятивный блок.** RSI + CVD + MFI говорят одно и то же → это сильнее, чем 3 отдельных сигнала. Считай как 5 голосов.

3. **Маршрут — одна ветка.** Никаких развилок. Ты выбрал направление — иди по нему.

4. **Каждая точка маршрута = уровень из расчётов.** Нельзя поставить произвольную цену.

5. **Слом — близкий и конкретный.** Не дальше 1.5 ATR_d. С обоснованием.

6. **Манипуляция — первый шаг маршрута, не альтернативный сценарий.** Задёрг → разворот → основной ход. Всё в одной ветке.

7. **Не добавляй уровни из головы.** Только то, что посчитано: пивоты, Fibo, VP, swing, FVG, паттерны.

8. **Зоны < 0.3% — одна точка.** Маршрут не засоряется.
